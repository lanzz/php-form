Class Form

Extends class Form_Container

This class represents the entire form. It is a Form_Container, because it contains the top-level Form_Element instances.

It provides several static methods to instantiate a Form from a given set of submitted values â€” either from actual GET/POST submitted values, or directly from an array of values.

Factory methods

These methods allow for instantiation of Form instances from various data sources

  • from_array(array $submission, $name = null)
  • from_get($context = null)
  • from_post($context = null)
  • from_request($context = null)

Formatting methods

These methods return the values of the form in various formats

  • hidden()
  • query()

Miscellaneous methods

  • is_submitted()
  • set_defaults(array $defaults)
