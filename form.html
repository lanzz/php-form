Class Form

This class represents the entire form. It is a Form_Container, because it contains the top-level Form_Element instances.

It provides several static factory methods to instantiate a Form from a given set of submitted values â€” either from actual GET/POST parameters, or directly from an array of values.

Extends class Form_Container

Factory methods

These methods create new Form instances populated with submitted values from various data sources

  • Form from_array(array $submission, string $name = null)
  • Form from_get(string $context = null)
  • Form from_post(string $context = null)
  • Form from_request(string $context = null)

Formatting methods

These methods return the values of the form in various formats

  • string hidden()
  • string query()

Miscellaneous methods

  • boolean is_submitted()
  • Form set_defaults(array $defaults)
  • array get_values()
