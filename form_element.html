Class Form_Element

This class represents an element in the form. It is just a value wrapper â€” it does not carry any information about the actual HTML element that should be rendered, and indeed can be rendered as any HTML form element, but also as a query string.

It can contain sub-elements, so it is an instance of Form_Container.

Extends class Form_Container

Accessor methods

These methods return various properties of the element

  • string get_name()
  • mixed get_default()
  • mixed get_submitted()
  • mixed get_value()
  • string get_type()
  • string get_id()

Formatting methods

These methods return the value of the element in various formats

  • string as_string()
  • string as_html()
  • string as_url()
  • string query()

Attribute rendering methods

These methods render various input element attributes

  • string name()
  • string value()
  • string id(string $prefix = null, string $suffix = null)
  • string checked(string $value)
  • string selected(string $value)

Input element rendering methods

These methods render entire <input> elements

  • string hidden()
  • string input(string $attributes = null)
  • string password(string $attributes = null)
  • string textarea(string $attributes = null)
  • string checkbox(string $value, string $attributes = null)
  • string radio(string $value, string $attributes = null)
  • string select(array $values, string $attributes = null)
  • string submit(string $label, string $attributes = null)
