Static method Form::from_get

Parameters:

  • string $name = null

Returns: Form

Creates a new Form instance, populated with submitted values from the URL query parameters.

$name specifies the point within the $_GET array from which to retrieve the form values, and determines the prefix for the form elements' names. For example:

$form = Form::from_get('foo[bar]');
$element = $form->baz;
echo $element->get_name(); // => 'foo[bar][baz]'

echo $element->get_submitted(); // will print the value of $_GET['foo']['bar']['baz']

If $name is not provided, the form will be populated with the full $_GET array, and will not have a name prefix:

$form = Form::from_get();
$element = $form->baz;
echo $element->get_name(); // => 'baz'

echo $element->get_submitted(); // will print the value of $_GET['baz']
