Static method Form::from_request

Parameters:

  • string $name = null

Returns: Form

Creates a new Form instance, populated with combined submitted values from the GET and POST parameters, and with $name as prefix for the form elements' names.

In case of name conflicts, POST parameters override GET parameters.

For more information, see the from_get() and from_post() methods.
