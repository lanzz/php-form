Method Form::set_defaults(array $defaults)

Sets default values for the form's elements.

These values are used as element's values if the form hasn't been submitted yet, or if there is no submitted value for an element (i.e. there is no matching key in the submitted values; a matching key with an empty string value is a valid submitted value).

Returns the form instance, allowing method chaining.
