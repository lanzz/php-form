Static method Form::from_post

Parameters:

  • string $name = null

Returns: Form

Creates a new Form instance, populated with submitted values from the POST parameters.

$name specifies the point within the $_POST array from which to retrieve the form values, and determines the prefix for the form elements' names. For example:

$form = Form::from_post('foo[bar]');
$element = $form->baz;
echo $element->get_name();

foo[bar][baz]

$element->get_submitted() will return the value of $_POST['foo']['bar']['baz']

If $name is not provided, the form will be populated with the full $_POST array, and will not have a name prefix:

$form = Form::from_post();
$element = $form->baz;
echo $element->get_name();

baz

$element->get_submitted() will return the value of $_POST['baz']
