Method Form->get_values

Parameters: none

Returns: array

Returns all form elements' values as an array.

$form = Form::from_array(array('foo' => 123, 'bar' => array('baz' => 123)));
print_r($form->get_values());
/* prints:
Array
  (
      [foo] => 123
      [bar] => Array
          (
              [baz] => 123
          )

  )
*/
