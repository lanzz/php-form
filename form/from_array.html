Static method Form::from_array

Parameters:

  • array $submission
  • string $name = null

Returns: Form

Creates a new Form instance, populated with submitted values from the $submission array.

$name sets a prefix for the form's elements' names:

$form = Form::from_array(array('baz' => 123), 'foo[bar]');
$element = $form->baz;
echo $element->get_name();

foo[bar][baz]

If $name is not provided, the form will have no name prefix:

$form = Form::from_array(array('baz' => 123));
$element = $form->baz;
echo $element->get_name();

baz
