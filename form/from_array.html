Factory method Form::from_array(array $submission, $name = null)

Instantiate a new form, populating it with submitted values from the $submission array, and setting its name to $name

The submitted values will be used in instantiating children elements. For example:

$form = Form::from_array(array('foo' => 123));
$element = $form->foo;
echo $element->get_value(); // => '123'

If there is no submitted value for an element, its get_value() method will return its default value instead; see the set_defaults() method.

The form's name sets a prefix for the form's elements' names. For example:

$form = Form::from_array(array('baz' => 123), 'foo[bar]');
$element = $form->baz;
echo $element->get_name(); // => 'foo[bar][baz]'

If $name is not provided, the form will have no name prefix:

$form = Form::from_array(array('baz' => 123));
$element = $form->baz;
echo $element->get_name(); // => 'baz'
