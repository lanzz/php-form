    Class representing the entire form.

    A container might be an entire form (the Form class is an instance of Form_Container), or it could be a form element containing child elements.

    Set an error for the container, clearing previous errors.

    set_error(string | null $error, string | null $code) : \Form
    Inherited

    All previous errors with the same $code will be cleared. The default error code is :default — if $code is omitted, only the messages for error code :default will be cleared and the new error will be added with a :default error code.

    If the $error parameter is null, only an error condition is raised for the specified error code, but no message is added.
