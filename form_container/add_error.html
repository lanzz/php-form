Method Form_Container->add_error

Parameters:

  • string $error
  • string $code = null

Returns: Form_Container

Adds an error message for the container. Previous error messages will be preserved.

If you pass $code, then the message will be specific to that error code. Error codes are independent namespaces, and messages added with add_error() to a specific namespace can later be cleared or retrieved together.

The default error code is :default.

$form->example->add_error('Lorem');
$form->example->add_error('Ipsum');
print_r($form->example->get_errors());
// Array
// (
//     [0] => Lorem
//     [1] => Ipsum
// )

$form->example->add_error('Dolor', 'foo');
$form->example->add_error('Sit Amet', 'foo');
print_r($form->example->get_errors());
// Array
// (
//     [0] => Lorem
//     [1] => Ipsum
//     [2] => Dolor
//     [3] => Sit Amet
// )

$form->example->clear_errors('foo');
print_r($form->example->get_errors());
// only "Dolor" and "Sit Amet" are cleared: Array
// (
//     [0] => Lorem
//     [1] => Ipsum
// )
