Method Form_Container->if_errors

Parameters:

  • string $string
  • string $code = null

Returns: string

Returns a given string only if errors have been added for the container, or an empty string otherwise.

If you pass $code, then if_errors() will only return the specified string if errors with that specific error code have been added.

Note that if you call if_errors() with no $code, then it will return the specified string if any errors have been added for the container. If you want to test only for errors with the default error code, you must invoke $form->if_errors($string, ':default') explicitly.

$form->example->add_error('Lorem', 'foo');

print($form->example->if_errors(' class="error" '));
// prints class="error"

print($form->example->if_errors(' class="error" ', 'xyzzy'));
// prints an empty string
