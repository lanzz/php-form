Property Access Magic Methods

  • Method Form_Container->__get()
  • Method Form_Container->__set()
  • Method Form_Container->__isset()
  • Method Form_Container->__unset()

The magic methods allow referencing child elements by name as instance properties.

$form = Form::from_array(array('foo' => 1, 'bar' => array(3, 4, 5)));

print($form->foo->get_value()."\n");
// prints "1"

print_r($form->bar->get_value());
// Array
// (
//     [0] => 3
//     [1] => 4
//     [2] => 5
// )

You can also update the submitted value for child elements:

$form = Form::from_array(array('foo' => 1));

print($form->foo->get_value()."\n");
// 1

$form->foo = 2;
print($form->foo->get_value()."\n");
// 2

Note that updating the submitted value of a parent will not have effect on child elements' values:

$form = Form::from_array(array('foo' => array('bar' => 1, 'baz' => 2)));

print($form->foo->bar->get_value()."\n");
// prints "1"

$form->foo = array('bar' => '3', 'baz' => 4);
print($form->foo->bar->get_value()."\n");
// still prints the old value "1"

Unsetting a property deletes the value and destroys the child element:

$form = Form::from_array(array('foo' => array('bar' => 1, 'baz' => 2)));
print_r($form->foo->get_value());
// Array
// (
//     [bar] => 1
//     [baz] => 2
// )

print($form->foo->bar->get_type()."\n");
// prints "integer"

unset($form->foo->bar);
print_r($form->foo->get_value());
// Array
// (
//     [baz] => 2
// )

print($form->foo->bar->get_type()."\n");
// prints "NULL"

Testing if a child element is set returns true only if the element has been submitted; an element might have a default value, but still will test as unset if no value has been submitted for it.

$form = Form::from_post();
if (isset($form->password)) {
	// the "password" element has been submitted
}
