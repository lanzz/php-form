Method Form_Container->has_errors

Parameters:

  • string $code = null

Returns: boolean

Test if any error messages have been added for the container.

If you pass $code, then has_errors() will only return true if errors with that specific error code have been added.

Note that if you call has_errors() with no $code, then it will return true if any errors have been added for the container. If you want to test only for errors with the default error code, you must invoke $form->has_errors(':default') explicitly.

$form->example->add_error('Lorem', 'foo');
$form->example->add_error('Ipsum', 'foo');
$form->example->add_error('Dolor', 'bar');
$form->example->add_error('Sit Amet', 'bar');

print($form->example->has_errors()? 'true': 'false');
// prints "true"

print($form->example->has_errors('foo')? 'true': 'false');
// prints "true"

print($form->example->has_errors('xyzzy')? 'true': 'false');
// prints "false"
