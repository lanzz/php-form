Iterator Interface Methods

  • Method Form_Container->next()
  • Method Form_Container->rewind()
  • Method Form_Container->valid()

The Iterator interface allows iterating over a Form_Container in a foreach loop.

$form = Form::from_array(array('bar' => 1, 'baz' => 2), 'foo');
foreach ($form as $key => $element) {
	print($key.": ".$element->name()."\n");
}

// bar:  name="foo[bar]"
// baz:  name="foo[baz]"
