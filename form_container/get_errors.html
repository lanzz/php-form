Method Form_Container->get_errors

Parameters:

  • string $code = null

Returns: array

Returns error messages previously added for the container.

If you pass $code, then only messages specific to that error code are returned.

Note that if you call get_errors() with no $code, then all error messages for all error codes will be returned in a flat array. If you want to retrieve only the messages for the default error code, you must invoke $form->get_errors(':default') explicitly.

$form->example->add_error('Lorem', 'foo');
$form->example->add_error('Ipsum', 'foo');
$form->example->add_error('Dolor', 'bar');
$form->example->add_error('Sit Amet', 'bar');

print_r($form->example->get_errors());
// Array
// (
//     [0] => Lorem
//     [1] => Ipsum
//     [2] => Dolor
//     [3] => Sit Amet
// )

print_r($form->example->get_errors('foo'));
// Array
// (
//     [0] => Lorem
//     [1] => Ipsum
// )

print_r($form->example->get_errors('bar'));
// Array
// (
//     [0] => Dolor
//     [1] => Sit Amet
// )
