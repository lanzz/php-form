Method Form_Container->set_error

Parameters:

  • string $error
  • string $code = null

Returns: Form_Container

Sets an error message for the container. Previous error messages will be cleared.

If you pass $code, then the message will be specific to that error code. Error codes are independent namespaces, and set_error() will only replace the messages for the same error code it is invoked with.

The default error code is :default.

$form->example->set_error('Lorem');
print_r($form->example->get_errors());
// Array
// (
//     [0] => Lorem
// )

$form->example->set_error('Ipsum');
print_r($form->example->get_errors());
// previous error "Lorem" is replaced: Array
// (
//     [0] => Ipsum
// )

$form->example->set_error('Dolor', 'foo');
print_r($form->example->get_errors());
// "Ipsum" is not removed, because "Dolor" has a different error code: Array
// (
//     [0] => Ipsum
//     [1] => Dolor
// )

$form->example->set_error('Sit Amet', 'foo');
print_r($form->example->get_errors());
// only "Dolor" is replaced: Array
// (
//     [0] => Ipsum
//     [1] => Sit Amet
// )
