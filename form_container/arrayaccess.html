ArrayAccess Interface Methods

The ArrayAccess interface allows referencing child elements by name or by index.

$form = Form::from_array(array('foo' => 1, 'bar' => array(3, 4, 5)));
print($form['foo']->get_value()."\n");
// prints "1"

print($form['bar'][2]->get_value()."\n");
// prints "5"

print_r($form['bar']->get_value());
// Array
// (
//     [0] => 3
//     [1] => 4
//     [2] => 5
// )

You can also update the submitted value for child elements:

$form = Form::from_array(array('foo' => 1));
print($form['foo']->get_value()."\n");
// 1

$form['foo'] = 2;
print($form['foo']->get_value()."\n");
// 2

Note that updating the submitted value of a parent will not have effect on already initialized child elements:

$form = Form::from_array(array('foo' => array('bar' => 1, 'baz' => 2)));
print($form['foo']['bar']->get_value()."\n");
// prints "1"

$form['foo'] = array('bar' => '3', 'baz' => 4);
print($form['foo']['bar']->get_value()."\n");
// still prints the old value "1"

print($form['foo']['baz']->get_value()."\n");
// prints the new value "4", because $form['foo']['baz'] was first accessed after the value of $form['foo'] was changed

Unsetting an index deletes the value, and destroys the child element, if it was already initialized:

$form = Form::from_array(array('foo' => array('bar' => 1, 'baz' => 2)));
print_r($form['foo']->get_value());
// Array
// (
//     [bar] => 1
//     [baz] => 2
// )

print($form['foo']['bar']->get_type()."\n");
// prints "integer"

unset($form['foo']['bar']);
print_r($form['foo']->get_value());
// Array
// (
//     [baz] => 2
// )

print($form['foo']['bar']->get_type()."\n");
// prints "NULL"
