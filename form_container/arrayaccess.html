ArrayAccess Interface Methods

  • Method Form_Container->offsetGet()

  • Method Form_Container->offsetUnset()
  • Method Form_Container->offsetExists()

The ArrayAccess interface allows referencing children elements by name or by index.

$form = Form::from_array(array('foo' => 1, 'bar' => array(3, 4, 5)));
print($form['foo']->get_value()."\n");
print($form['bar'][2]->get_value()."\n");
print_r($form['bar']->get_value());

// 1
// 5
// Array
// (
//     [0] => 3
//     [1] => 4
//     [2] => 5
// )
