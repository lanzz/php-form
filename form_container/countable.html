Countable Interface Method

  • Method Form_Container->count

Returns the count of elements in the container for count($container) invocations.

Counting child elements — each submitted or default value is counted:

$form = Form::from_array(array('foo' => 123, 'bar' => 456));
$form->set_defaults(array('foo' => 789, 'baz' => null));
echo count($form);
// prints 3 — distinct child elements 'foo', 'bar' and 'baz'

Counting array values — submitted array overrides the default array:

$form = Form::from_array(array('foo' => array(3, 4, 5)));
$form->set_defaults(array('foo' => array(1)));
print(count($form->foo)."\n");
// prints 3 — the submitted list (1, 2, 3) overrides the default 1-element list (1)
