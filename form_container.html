Abstract class Form_Container

Implements interface ArrayAccess

Implements interface Countable

Implements interface Iterator

This abstract class is the base for both the Form and the Form_Element classes. It represents a container for named form elements; both the actual form and each form element in it can contain children Form_Element instances.

If a non-existent name is accessed, a new Form_Element instance will be automatically created for the new element.

Property access methods

These methods give access to children elements as $container->child_name

  • Form_Element __get(string $name)
  • void __set(string $name, mixed $value)
  • boolean __isset(string $name)
  • void __unset(string $name)

Countable interface methods

Return number of the children elements when invoking count($element)

  • int count()

Iterator interface methods

These methods allow for foreach iteration over the children elements

  • Form_Element current()
  • string key()
  • void next()
  • void rewind()
  • boolean valid()

ArrayAccess interface methods

These methods give access to children elements as $container['child_name'] or $container[7]

  • Form_Element offsetGet(mixed $index)
  • void offsetSet(mixed $index, mixed $value)
  • boolean offsetExists(mixed $index)
  • void offsetUnset(mixed $index)

Error handling methods

These methods allow setting and checking for error messages for the container

  • Form_Container clear_errors(string $code = null)
  • Form_Container set_error(string $error, string $code = null)
  • Form_Container add_error(string $error, string $code = null)
  • boolean has_error(string $code = null)
  • array get_errors(string $code = null)
  • string if_error(string $return, string $code = null)
