Abstract class Form_Container

This abstract class is the base for both the Form and the Form_Element classes. It represents a container for named form elements; both the actual form and each form element in it can contain children Form_Element instances.

If a non-existent name is accessed, a new Form_Element instance will be automatically created for the new element.

Implements interface Countable

Return number of the children elements when invoking count($element).

  • int count()

Implements interface Iterator

–êllow foreach iteration over the children elements.

Implements interface ArrayAccess

Allow access to children elements as $container['child_name'] or $container[7].

  • Form_Element offsetGet(mixed $index)
  • void offsetSet(mixed $index, mixed $value)
  • boolean offsetExists(mixed $index)
  • void offsetUnset(mixed $index)

Property access methods

Allow access to children elements as $container->child_name.

  • Form_Element __get(string $name)
  • void __set(string $name, mixed $value)

  • void __unset(string $name)
